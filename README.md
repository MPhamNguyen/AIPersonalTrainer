# AIPersonalTrainer Chatbot

An intelligent Health and Fitness backend that leverages the Wger Exercise Database and Google Gemini 3.0 to provide personalized workout routines, real-time coaching via chat, and algorithmic progressive overload tracking.

## 🚀 Quick Start

### 1. Installation
Ensure you have Python 3.9+ installed. First, install the required dependencies:

```bash
pip install -r requirements.txt
```

### 2. Start the local server

```bash
uvicorn main:app
```

### 3. To access page navigate to

```bash
http://127.0.0.1:8000/docs
```

### Frequent Troubleshoot

1. Make sure you have a Google Gemini API key either hardcoded or as an environmental variable.
